Nova Browser Agent

The official documentation for the world's first true Agentic Browser Extension.

Version: 1.0.0 | Stack: React, Chrome Extension V3, LLM

What is Nova?
Nova is an intelligent layer capable of autonomously navigating the web. It doesn't just "read" pages; it clicks, types, scrolls, and analyzes content to perform complex tasks like shopping research or filling forms.

Get up and running in less than 5 minutes.

1. Installation

Run these commands in your terminal:

# Clone the repo
git clone https://github.com/your-org/rocket-extension.git

# Install dependencies
cd rocket-extension
npm install

# Build production bundle
npm run build

2. Load Extension

  1. Go to chrome://extensions.
  2. Enable Developer Mode (top right).
  3. Click Load unpacked and select the dist/ folder.
Pro Tip: Pin the extension to your toolbar for easy access. The side panel is persistent!

This section explains the "Magic" for engineers and technical interviewers.

1. The Core Loop: AgentService

The AgentService is the central orchestrator. It uses a "Thick Client" architecture where logic runs locally in the browser background.

Simplified Logic Flow:

class AgentService {
  async runTask(userGoal) {
    // 1. Intent Detection using micro-LLM
    const intent = await IntentDetector.analyze(userGoal);
    
    if (intent === 'SHOPPING') {
        return await ShoppingService.deepResearch(userGoal);
    }

    // 2. Loop until goal is met
    while (!taskComplete) {
        // A. SEE: Capture DOM Snapshot
        const state = await Observer.getBrowserState();
        
        // B. THINK: Ask LLM for next action
        const action = await LLM.decide(state, userGoal);
        
        // C. ACT: Execute via Debugger API
        await HybridCore.execute(action);
        
        // D. VERIFY: Self-Healing Check
        if (hasError) await RecoveryEngine.fix(action);
    }
  }
}

2. The Vision System: SemanticDOM

We solve the issue of "messy HTML" by converting the DOM into a cleaning Semantic Map.

  • Filtering: Removes scripts, ads, and invisible nodes.
  • Clustering: Groups related elements (e.g., Image + Title + Price = "Product Card").

What the AI actually sees (JSON Representation):

3. Resilience: RecoveryEngine

The Problem: Web pages are dynamic. An element ID might change between the "Think" and "Act" phases.

The Solution: If a click fails, the Recovery Engine kicks in.

async function recover(failedAction) {
    // Strategy 1: Fuzzy Match Text
    // "Buy Now" (ID 45) is gone? Look for any button with "Buy Now".
    const newId = await DOM.findClosestMatch(failedAction.targetText);
    
    if (newId) return performAction(newId);
    
    // Strategy 2: Re-plan
    return await LLM.replanningStrength(failedAction);
}

üõçÔ∏è Feature: Deep Research

Nova doesn't just search; it researches. Here is the workflow for the "Shopping Agent":

  1. Constraint Parsing: Extracts "Budget: < 20k", "Year: 2025" from user prompt.
  2. Knowledge Gathering: Reads "Best of" blog posts to find top models.
  3. Multi-Site Price Check: Spiders across Amazon, Flipkart, and Croma simultaneously.
  4. Deal Analysis: Calculates final price including bank offers.
